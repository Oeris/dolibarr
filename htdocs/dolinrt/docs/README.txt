Ce module permet :

- Création d'un tiers et d'un contact Dolibarr depuis les champs personalisés RT
- Récupération d'un contact Dolibarr dans les champs personalisés RT
- Récupération d'une proposition commerciale Dolibarr et de son contenu dans RT
- Modification de la proposition commerciale
- Création et modification d'une facture Dolibarr depuis RT
- Téléchargement de la facture Dolibarr depuis RT
- Possibilité d'ajouter un forfait de facturation a une facture Dolibarr sur changement de statut (scrip RT)
- Possibilité d'ajouter à une facture le temps passé sur RT sur changement de statut (scrip RT)
- Possibilité de classer une facture au statut payé sur changement de statut (scrip RT) 

Compatible Dolibarr 5.0 et RT 4.1

INSTALLATION : La procédure d'installation se situe dans le répertoire docs du module.

SUPPORT : le support n'est pas inclus avec le module.
